# 開発（ローカル実行）

このドキュメントは開発者向けです。利用者としては GitHub Pages 上の公開サイトを操作してください（README 参照）。

## 必要環境

- Node.js
- npm

## セットアップ

```bash
npm install
```

## 開発サーバー

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## ローカルプレビュー

```bash
npm run preview
```

## デプロイ

GitHub Pages へのデプロイは [docs/deploy.md](docs/deploy.md) を参照してください。
