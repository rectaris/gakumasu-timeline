# 操作マニュアル

このマニュアルは、タイムラインアプリの操作方法をまとめたものです。

## 画面構成

- **固定ヘッダー**: 画面上部に固定表示
  - 左: メニュー（☰）、マニュアル（？）
  - 中央: タイトル
- **左メニュー**: カテゴリとレーンの表示切り替え
- **タイムライン**: レーンごとのイベントを時系列に表示
- **詳細パネル**: 右側にイベント詳細を表示

## メニュー操作

1. ヘッダー左の **☰** をクリックしてメニューを開きます。
2. **カテゴリ** を選択します。
3. **表示レーン** のチェックで表示/非表示を切り替えます。
4. **一括** チェックでカテゴリ内の全レーンをまとめて選択/解除できます。
5. 背景クリック、または **Escape** キーでメニューを閉じます。

## タイムライン操作

- **イベント選択**: バーをクリックすると右の詳細パネルに内容が表示されます。
- **ツールチップ**: マウスオーバーで `title` と `detail` が表示されます。
- **詳細パネルを閉じる**: 右上の × または **Escape** キー。

## ズーム操作

- **＋ / − ボタン**でズーム段階を変更
  - 全期間 → 年 → 月 → 日
- **スライダー**で中心の年/月/日を移動
- **< / > ボタン**で 1 単位ずつ移動（長押し対応）
- **キーボード**
  - ← / → : 年・月・日を 1 単位移動
  - Shift + ← / → : 年単位で移動
- **スワイプ**: タイムライン上で左右スワイプ

## 既知の仕様

- 年/全期間は月単位、月/日は日単位で描画されます。
- 実暦は考慮せず、1か月=31日として扱います。
